# 🏭 Фабрика Додатків UNO-LINE

Мобільний PWA-застосунок (Progressive Web App) для генерації, виконання та збереження автономних веб-інструментів за допомогою Gemini API. 

Проект спроектовано як "Командний центр" для швидкого створення інженерних калькуляторів, технічних довідників та інших утиліт безпосередньо зі смартфона, без використання бекенду чи сторонніх серверів.

## ⚙️ Основні можливості

* **Генерація коду на льоту:** Створення готових HTML/JS/CSS віджетів через текстовий запит до ШІ (моделі Gemini 3).
* **Локальне сховище (Офлайн-режим):** Збережені додатки записуються в пам'ять пристрою і працюють без доступу до інтернету.
* **Zero-Server Архітектура:** Вся логіка виконується виключно на клієнтському боці (у браузері).
* **Ізоляція та безпека (Sandbox):** Згенерований ШІ код запускається у безпечному контейнері `<iframe>`.
* **Локальне зберігання API:** Ключ доступу до Gemini ніде не публікується і зберігається лише у браузері кінцевого користувача.

## 📂 Структура проекту

Проект базується на класичній тріаді PWA, розміщеній у кореневій папці:
* `index.html` — основний інтерфейс, логіка запитів до API та робоча зона рендерингу.
* `sw.js` — Service Worker, що кешує оболонку застосунку для швидкого запуску офлайн.
* `manifest.json` — маніфест, що дозволяє системі Android ідентифікувати веб-сторінку як нативний додаток.
* `icon.png` — іконка застосунку (512x512).

## 📱 Встановлення та використання на смартфоні

1. Відкрийте згенероване посилання у браузері (рекомендовано Chrome) на Android-пристрої.
2. Браузер автоматично запропонує **"Додати на головний екран"** (Add to Home Screen). Якщо повідомлення не з'явилося, виберіть цю опцію вручну через меню браузера (⋮).
3. Запустіть "Фабрику Додатків" з робочого столу смартфона.
4. Відкрийте ліве випадаюче меню (⚙️) та вставте свій **Gemini API Key** (можна отримати в Google AI Studio).
5. Введіть завдання у рядок запиту та натисніть "Створити".

## 🛠 Стек технологій
* HTML5 / CSS3 / Vanilla JavaScript
* Google Gemini API (`gemini-1.5-flash` / `gemini-1.5-pro`)
* Service Worker API / LocalStorage API
